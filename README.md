# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025)

## How to run
1. Add input file named `input.txt` to the dir of the task you want to run. E.g. `internal/day1/input.txt`.
2. Run task by using command `go run cmd/main.go <task>`. E.g. `go run cmd/main.go 1a`.
